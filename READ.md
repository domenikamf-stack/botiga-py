# Proyecto de Optimización - Tienda Python
El objetivo de este proyecto es aprender a "limpiar" código (refactorizar), 
documentarlo y gestionarlo con control de versiones (Git).

## Funcionalidades
- Poder calcular los precios con descuento  por cada posicion
- Tener descuentos para clientes que sean menores de 18 años.

## Refactorizaciones aplicadas
1. Rename: renombrar a las variables por nombres mas claros (`p` -> `precio_base`).
2. Magic Numbers: Uso de nombres especificos para porcentajes y edades.
3. Extract Method: Dividir el codigo por funciones

## Cómo ejecutar
- Tener  instalado python y crear un archivo con el nombre: `python botiga.py`
- Escribir el codigo anterior y ejecutar lo en la terminal
